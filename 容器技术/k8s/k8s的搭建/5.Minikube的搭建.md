# Minikube的搭建

k8s的安装还是挺繁琐的，Minikube提供一个轻量化和便携的k8s实现，它的核心目标是让你能够在本地计算机上轻松运行和体验k8s,拿来做一些测试，学习……还是挺不错的。

官网：https://minikube.sigs.k8s.io/docs/

中文：https://minikube.kubernetes.ac.cn/docs/

## 使用docker安装

Minikube支持使用docker进行运行。



## windos上安装

使用Windows自带的包管理工具``winget``可以进行一键安装(需要docker)

```PowerShel
winget install minikube
```

启动集群

```PowerShel
minikube start --driver=docker
```
