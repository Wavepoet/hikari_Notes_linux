# **Kubernetes Dashboard安装**

---

Dashboard是基于k8s的图形化界面。是Kubernetes 的官方提供的 Web UI。

个人感觉装挺简单的

github：

[GitHub - kubernetes/dashboard: General-purpose web UI for Kubernetes clusters](https://github.com/kubernetes/dashboard)

请更具自己的k8s查看github版本进行安装，我在这里安装的k8s是v1.28.2版本，选择了**Dashboard** v2.7安装。

---

拉去yaml

```bash
wget https://raw.githubusercontent.com/kubernetes/dashboard/v2.7.0/aio/deploy/recommended.yaml
```

编制yaml，暴露端口。

```bash
 37   name: kubernetes-dashboard
 38   namespace: kubernetes-dashboard
 39 spec:
 ##在此增加type: NodePort
 40   type: NodePort
 41   ports:
 42     - port: 443
 43       targetPort: 8443

```

安装**Dashboard（请耐心等待）**

```bash
kubectl apply -f recommended.yaml
```

创建用户并设置管理员权限

```bash
kubectl create serviceaccount dashboard-admin -n kubernetes-dashboard
kubectl create clusterrolebinding dashboard-admin-rb --clusterrole=cluster-admin --serviceaccount=kubernetes-dashboard:dashboard-admin
```

获取令牌（保存好）

```bash
kubectl create token dashboard-admin -n kubernetes-dashboard
```

查看端口

```bash
kubectl get svc  -n kubernetes-dashboard -w
#找到kubernetes-dashboard的映射端口，例如443:31293映射的宿主机端口就是31293
```

访问：``http://<IP>:<映射端口>``。由于使用自签名 SSL 证书，浏览器可能发出安全警告。登录需要使用令牌，复制粘贴即可。
