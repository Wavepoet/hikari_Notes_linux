# Docker镜像加速器

因为一些原因，国内无法访问Docker Hub或者访问的速度非常慢，因此需要配置Docker镜像加速器来提高拉取镜像的速度。**如果有代理的话可以直接使用代理访问Docker Hub即可**。

编写`/etc/docker/daemon.json`文件，没有文件则创建。

```bash
cat /etc/docker/daemon.json <<EOF
{"registry-mirrors":["[https://docker.xuanyuan.me](https://docker.xuanyuan.me/)"]}
EOF
```

编写完成后重新docker服务。

```bash
sudo systemctl daemon-reload
sudo systemctl restart docker
```

输入`docker info`查看加速器是否配置成功，成功则会输出以下字段：

```bash
[root@localhost docker]# docker info
 …………
 Registry Mirrors:
  https://docker.xuanyuan.me/
 Live Restore Enabled: false
```

[`https://docker.xuanyuan.me`](https://docker.xuanyuan.me/)为代理网址，可以尝试使用`docker pull`拉取镜像，拉取失败的话可以换一个代理网站。

```bash
  https://docker.xuanyuan.me/
  https://docker.m.daocloud.io/
  https://docker.1ms.run/
  https://docker.1panel.live/
```
