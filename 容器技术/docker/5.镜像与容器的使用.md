# 镜像与容器的使用

## 列出本地容器

使用 `docker images` 命令可以列出本地镜像。

```bash
[root@localhost qyc]# docker images
REPOSITORY   TAG       IMAGE ID       CREATED       SIZE
nginx        latest    ad5708199ec7   2 weeks ago   192MB
ubuntu       latest    e0f16e6366fe   4 weeks ago   78.1MB
centos       centos8   5d0da3dc9764   3 years ago   231MB

-----------------------------------------------------------
镜像的仓库源  镜像的标签  镜像ID         镜像创建时间   镜像大小
```

常用参数：

`docker image prune`       #清理所有悬空镜像（无标签的`<none>` 镜像）
`-a`      #显示所有镜像

`-f`      #进行筛选

`--digests`      #显示摘要

`-q`      #仅仅显示镜像ID

## 从镜像源拉取镜像

使用  `docker pull`从镜像源拉取镜像，默认镜像源为Docker Hub，如果不指定版本则默认拉取latest（最新）。

```bash
[root@localhost qyc]# docker pull php
Using default tag: latest
latest: Pulling from library/php
……
Digest: sha256:24a6d741b50242e8a9bd8eb14345790e5266eb8314c528a254ce8c014e961e9d
Status: Downloaded newer image for php:latest
docker.io/library/php:latest

[root@localhost qyc]# docker images
REPOSITORY                         TAG       IMAGE ID       CREATED        SIZE
php                                latest    797fe7681a3e   44 hours ago   545MB
nginx                              latest    ad5708199ec7   2 weeks ago    192MB
ubuntu                             latest    e0f16e6366fe   4 weeks ago    78.1MB
centos                             centos8   5d0da3dc9764   3 years ago    231MB

```

常用参数：

**`-t`     #**在新容器内指定一个伪终端或终端。

**`-i`     #**允许你对容器内的标准输入 (STDIN) 进行交互。

`-d`     #后台运行
`name=”name”`      #命名容器名字

`-p`     #指定容器使用的端口

## 启动一个容器

使用`docker run`来启动一个容器，如果不指定版本则默认使用latest（最新）。

```bash
[root@localhost qyc]# docker run centos:centos8 /bin/echo "Hello world"
Hello world
```

解析

`centos:centos8`：指定使用centos:centos8镜像。centos8是镜像的Tag。

`/bin/echo "Hello world"`：启动容器后在容器内执行的命令。

### 尝试使用交互式运行

```bash
[root@localhost qyc]# docker run -it centos:centos8 /bin/bash
[root@402ab9df4b1f /]#
[root@402ab9df4b1f /]# pwd
```

## 查看正在运行的容器

使用`docker ps`查看正在运行的容器

```bash
[root@localhost qyc]# docker ps
CONTAINER ID   IMAGE            COMMAND                   CREATED          STATUS          PORTS     NAMES
5fdf53a9fd05   centos:centos8   "/bin/sh -c 'while t…"   21 seconds ago   Up 20 seconds             helloworld

----------------------------------------------------------------------------------------------------------------
容器ID         镜像             命令                      创建时间          容器状态         端口       名字
```

容器状态：

状态有7种：

- created（已创建）
- restarting（重启中）
- running 或 Up（运行中）
- removing（迁移中）
- paused（暂停）
- exited（停止）
- dead（死亡）

## 停止容器

使用`docker stop` 来停止容器。

```bash
[root@localhost qyc]# docker stop 5fdf53a9fd05
5fdf53a9fd05
[root@localhost qyc]# docker ps
CONTAINER ID   IMAGE     COMMAND   CREATED   STATUS    PORTS     NAMES
```
